Esta Aplicação tem como finalidade a correção monetária pela SELIC e UFFI. 
Dado o Valor a ser corrigido, as datas de Vencimento e Pagamento, ela verifica as datas e as correções necessárias para aplicar a SELIC mensal e a UFFI. Trazendo os valores corrigidos e memorial incidente sobre cálculo. 
As Selics mensais e UFFIs anuais estão inseridas até a presente Data. Onde podem ser verificadas e adicionados novos valores correntes no momento da utilização caso não tenha um valor desejado.
Esta Aplicação foi desenvolvida, monoliticamente,  em c# .net 9 utilizando o entity frameworkcore integrando-se ao SQLite embutido na própria aplicação. 
Sua interface é provida pelo Blasor Server e FluentUI.
Esta aplicação, não acessa recursos externos, apenas os inseridos nela própria, para sua utilização.
Foi desenvolvida com a finalidade de atender a demanda interna de servidores da SMFO, para auxilia-los nos valores a serem lançadas em outros sistemas de informação.
